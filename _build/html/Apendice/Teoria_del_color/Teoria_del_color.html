Teoria del color#

Introducci√≥n#

Para nuestros ojos, una fuente de luz tiene coloraci√≥n blanca cuando la intensidad de cada frecuencia del espectro visible es, aproximadamente, constante.

El mejor ejemplo es el sol.

Espacio de colores CIE RGB#

Los colores aparecen cuando el espectro de la fuente no est√° ditribuido de forma uniforme. Nuestros ojos, encargados de intrepretar este espectro como color, poseen c√©lulas fotoreceptoras o conos celulares sensibles a los colores: rojo (R), verde (G) y azul (B)

Matem√°ticamente, existen diversas formas de transformar un espectro, \(I_\mathrm{spec}(\lambda)\), a c√≥digo de colores RGB. En este curso, usamos el m√≥dulo ref_spectra.color_system de la librer√≠a empylib, el cual est√° basado en Comisi√≥n internacional de la iluminaci√≥n (CIE)

En t√©rminos simples, el procedimiento consiste en determinar los valores triest√≠mulo \(X\), \(Y\) y \(Z\) a partir de las funciones de combinaci√≥n de colores \(\bar{x}(\lambda)\), \(\bar{y}(\lambda)\) y \(\bar{z}(\lambda)\), que representan la precepci√≥n visual humana de forma num√©rica:

\[\begin{equation*} X = \int_{0}^\infty I_\mathrm{spec}(\lambda)\bar{x}(\lambda) d\lambda,\quad\quad Y = \int_{0}^\infty I_\mathrm{spec}(\lambda)\bar{y}(\lambda) d\lambda,\quad\quad Z = \int_{0}^\infty I_\mathrm{spec}(\lambda)\bar{z}(\lambda) d\lambda, \end{equation*}\]

Analicemos el comportamiento de las funciones de combinaci√≥n de colores.

Posterior a esto, el c√≥digo de colores \(\mathrm{RGB}\) se determina a partir del sistema:

\[\begin{equation*} \begin{bmatrix} 0.187 & 0.063 & 0.064 \\ 0.092 & 0.212 & 0.025 \\ 0 & 0.024 & 0.334 \\ \end{bmatrix}\left[\begin{array}{c} \mathrm{R} \\\mathrm{G} \\\mathrm{B} \end{array}\right] = \left[\begin{array}{c} x \\y \\z\end{array}\right] \end{equation*}\]

donde \(xyz\) est√°n dados por:

\[\begin{equation*} x = \frac{X}{X+Y+Z},\quad\quad\quad y = \frac{Y}{X+Y+Z},\quad\quad\quad z = \frac{Z}{X+Y+Z}, \end{equation*}\]

Modelos de colores#

Coloraci√≥n aditiva#

Es un modelo que explica la obtenci√≥n de un color a partir de la suma de los componentes primarios, com√∫nmente, rojo, verde y azul. La coloraci√≥n aditiva es el m√©todo utilizado en monitores electr√≥nicos, como LCD, LED, oLED, etc.

Analicemos el siguiente modelo de colores aditivos.

# importamos librer√≠as necesarias
import empylib as em
import empylib.nklib as nk
import empylib.ref_spectra as rf
import empylib.waveoptics as wv
import matplotlib.pyplot as plt
import numpy as np

def plot_additive_color(R,G,B):
    # lienzo para la figura
    fig, ax = plt.subplots()          
    fig.set_size_inches(6, 4)         # Tama√±o del gr√°fico
    plt.rcParams['font.size'] = '14'  # tama√±o de  fuente

    # Espectro de longitudes de onda entre 300 y 800 nm
    lam = np.linspace(0.3,0.8,500)

    # Generamos un material con emisividad en rojo, verde y azul
    wn = em.convert_units(np.array([450, 520, 630]),'nm','eV')
    wp = [B*0.02,G*0.02,R*0.02]  # Frecuencia wp (eV)
    T  = 0.05    # Taza de decaimiento (eV)

    eps_mat = complex(1,0)
    for i in range(len(wn)):
        # constante diel√©ctrica seg√∫n modelo de Lorentz
        eps_mat = eps_mat + (nk.lorentz(0,wp[i],wn[i],T,lam))**2

    N_mat = np.sqrt(eps_mat)

    Rw, Tw = wv.incoh_multilayer(lam,0, (1.0,N_mat,1.0), 200)
    Ew = 1 - Rw - Tw;

    # Graficamos el espectro de emisi√≥n
    ax.plot(lam*1E3,Ew*100)
    ax.set_ylabel('Emisividad (%)')
    ax.set_xlabel('Longitud de onda ($\mu$m)')
    ax.set_ylim(0,100)

    # interpretacion de colores
    Dcircle = 0.30
    ax2 = fig.add_axes([0.63,0.55, Dcircle, Dcircle])
    html_rgb = cs.spec_to_rgb(Ew*rf.AM15(lam),lam, out_fmt='html')
    Circle = plt.Circle((0, 0), Dcircle, color=html_rgb)
    ax2.add_patch(Circle)
    ax2.set_aspect('equal')
    ax2.set_xlim(-Dcircle*1.2,Dcircle*1.2)
    ax2.set_ylim(-Dcircle*1.2,Dcircle*1.2)
    ax2.set_xticks([])
    ax2.set_yticks([])
    ax2.set_facecolor('k');

@interact(R=(0,1,0.1), G=(0,1,0.1), B=(0,1,0.1))

Coloraci√≥n sustractiva#

Se basa en la formaci√≥n de colores a partir de la sustracci√≥n de colores primarios. Com√∫nmente, se utilizan los colores amarillo, cian y magenta.

El modelo, explica la coloraci√≥n de muchas frutas, verduras y plantas. Adem√°s, es la t√©cnica utilizada en impresi√≥n a color, fotograf√≠a y cinematograf√≠a.

El siguiente modelo explica el concepto de coloraci√≥n sustractiva

# importamos librer√≠as necesarias
import empylib as em
import empylib.nklib as nk
import empylib.ref_spectra as rf
import empylib.waveoptics as wv
import matplotlib.pyplot as plt
import numpy as np

def plot_substractive_color(R,G,B):
    # lienzo para la figura
    fig, ax = plt.subplots()          
    fig.set_size_inches(6, 4)         # Tama√±o del gr√°fico
    plt.rcParams['font.size'] = '14'  # tama√±o de  fuente

    # Espectro de longitudes de onda entre 300 y 800 nm
    lam = np.linspace(0.3,0.8,100)

    # Generamos un material con emisividad en rojo, verde y azul
    wn = em.convert_units(np.array([450, 520, 630]),'nm','eV')
    wp = [B*0.02,G*0.02,R*0.02]  # Frecuencia wp (eV)
    T  = 0.05                    # Tasa de decaimiento (eV)
    
    eps_mat = complex(1,0)
    for i in range(len(wn)):
        # constante diel√©ctrica seg√∫n modelo de Lorentz
        eps_mat = eps_mat + (nk.lorentz(0,wp[i],wn[i],T,lam))**2

    N_mat = np.sqrt(eps_mat)

    Rw, Tw = wv.incoh_multilayer(lam,0, (1.0,N_mat,1.0), 200)
    Ew = 1 - Rw - Tw
    Rw = (1 - Ew)

    # Graficamos el espectro de emisi√≥n
    ax.plot(lam*1E3,(1 - Ew)*100)
    ax.set_ylabel('Reflectancia (%)')
    ax.set_xlabel('Longitud de onda ($\mu$m)')
    ax.set_ylim(0,100)

    # interpretacion de colores
    Dcircle = 0.30
    ax2 = fig.add_axes([0.63,0.55, Dcircle, Dcircle])
    html_rgb = cs.spec_to_rgb(Rw*rf.AM15(lam),lam, out_fmt='html')
    Circle = plt.Circle((0, 0), Dcircle, color=html_rgb)
    ax2.add_patch(Circle)
    ax2.set_aspect('equal')
    ax2.set_xlim(-Dcircle*1.2,Dcircle*1.2)
    ax2.set_ylim(-Dcircle*1.2,Dcircle*1.2)
    ax2.set_xticks([])
    ax2.set_yticks([])
    ax2.set_facecolor('k');
    
    plt.show()
